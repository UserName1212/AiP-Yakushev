# Лабораторные работы по предмету "Алгоритмизация и программирование"

Репозиторий включает набор лабораторных работ, которые выполнены в рамках учебной практики по курсу "Алгоритмизация и программирование". Эти работы на языке C демонстрируют основные концепции программирования: указатели, массивы, структуры, рекурсия, работа с файлами и тестирование.

## Лабораторные работы

### Лабораторная работа 1
Базовые концепции программирования на C.

### Лабораторная работа 2
Работа с заголовочными файлами и модульной организацией кода.

### Лабораторная работа 3
Продвинутые темы в C.

### Лабораторная работа 4
Указатели, массивы, рекурсия, функции с переменным числом параметров и тестирование. 

### Лабораторная работа 5
Структуры и базовые операции с ними.

### Лабораторная работа 6
Картотека книг с использованием структур, динамической памяти, файлов и сортировки. 

## Компиляция и запуск

Каждая лабораторная работа комптлируется отдельно 
Код написан в программе VS 2022, в дирректории каждого проекта есть файл решения VS 2022, для простой компиляции и тестирования
